🚀 Expense Analytics Dashboard

A responsive React-based web application designed to track, analyze, and visualize personal expenses in an interactive dashboard format.

🔹 Features

Add, edit, and delete expenses

Categorize expenses (Food, Travel, Bills, etc.)

Dynamic expense analytics and summaries

Conditional rendering for empty states

Loading and error handling for API integration

Fully responsive UI

🛠 Tech Stack

React (Functional Components & Hooks)

JavaScript (ES6+)

REST API Integration

HTML5 & CSS3

Git & GitHub

🧠 Key Concepts Implemented

useState & useEffect for state management

Component-based architecture

Conditional rendering

API data fetching

Dynamic UI updates

📂 Project Structure

Reusable components folder

Separate API handling logic

Clean and modular code organization
